Abstract

Chat App is a concept real-time chatting application. It uses a HTML and Javascript in the front-end. In the backend, it uses PHP and MySQL. PHP cleans the string input, queries the database and builds the HTML. The Javascript calls the PHP every second to make it real-time. Users can login, logout, search for users, and chat.
