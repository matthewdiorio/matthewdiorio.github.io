Whale Shark API

A RESTful API to get facts about whale sharks.

  • Home
  • About

Abstract

A concept API to get facts about on whale sharks. Facts can be random or called specifically. Facts can also be added. The API was created using Node.JS and Express. A quick front-end shows a random whale shark fact along with subtle animations.


See it live here.


Usage


Base URL:

https://whalesharkapi.herokuapp.com

Get all facts:

Returns all instances of whale shark facts along with source.

GET whalesharkfacts/
Example Response: Status Code 200 

  [
   {
      "_id": "603d5af10437c265d2c91bb1",
      "fact": "Whale sharks are the worlds biggest fish!",
      "source": "https://www.example.com",
      "createdDate": "2021-02-23T01:31:29.564Z",
      "__v": 0
    }
  ]
Get a random fact:

Returns returns a random fact.

GET whalesharkfacts/random
Example Response: Status Code 200 

  [
   {
      "_id": "603d5af10437c265d2c91bb1",
      "fact": "Whale sharks are the worlds biggest fish!",
      "source": "https://www.example.com",
      "createdDate": "2021-02-23T01:31:29.564Z",
      "__v": 0
    }
  ]
Get a specific fact:

Returns all instances of whale shark facts along with source.

GET whalesharkfacts/{ID}
Name Type Required Description
ID String True ID of the fact you want to fetch.
Example Response: Status Code 200 

  [
   {
      "_id": "603d5af10437c265d2c91bb1",
      "fact": "Whale sharks are the worlds biggest fish!",
      "source": "https://www.example.com",
      "createdDate": "2021-02-23T01:31:29.564Z",
      "__v": 0
    }
  ]
Posting a new fact:

Post a new whale shark fact.

POST whalesharkfacts/
Name Type Required Description
fact String True The whale shark fact you want to post.
source String True The source of the fact.
Example Request: JSON 

  {
    "fact": "Whale sharks are the world's biggest fish!",
    "source": "https://www.example.com/"
  }
Example Response: Status code 201 

  {
    "_id": "604ad61c2c779e6179a2e788",
    "fact": "Whale sharks are the world's biggest fish!",
    "source": "https://www.example.com/",
    "createdDate": "2021-03-12T02:46:52.199Z",
    "__v": 0
  }
Change a fact:

Change one part or all of a whale shark fact information

PATCH whalesharkfacts/{ID}
Name Type Required Description
ID String True ID of the fact you want to update.
Example Request: JSON 

  {
    "fact": "Whale sharks really are the world's biggest fish!"
  }
Example Response: Status code 201 

  {
    "_id": "604ad61c2c779e6179a2e788",
    "fact": "Whale sharks really are the world's biggest fish!",
    "source": "https://www.example.com/",
    "createdDate": "2021-03-12T02:46:52.199Z",
    "__v": 0
  }
Delete a fact:

Delete a whale shark face with using it's assigned ID.

DELETE whalesharkfacts/{ID}
Name Type Required Description
ID String True ID of the fact you want to delete.
Example Response: Status Code 200 

  {
    "message": "Deleted Fact"
  }
